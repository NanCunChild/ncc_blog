多年以后，当 NCC 再次盯着日志里那行永远无法复现、也找不到原因的幽灵报错时，总会回想起那个风雨交加的凌晨。那是他在公司高楼的夜灯前，听着城市深处沉闷的雷声，加班 Debug 的时刻——也是一切荒诞的起点。

在那一天之前，所谓的“进程”、“内核”与“程序”，在 NCC 眼中不过是硅片上跳动的电子，是晶体管里生灭的比特位。NCC尚未见识过那个世界里猩红的血肉肌骨，也还未领教过它那绝对理性的冷漠，与深不见底的平庸之恶。

​那个晚上，NCC 只是一个被逼到崩溃边缘的普通人，正试图和一座存活了二十八年的 C++ 屎山进行绝望的肉搏。它的最老的一批文件，修改时间停留在 1998 年，历经两次服务器环境搬迁，被无数不知名的员工和实习生蹂躏过。如今，它已经畸变成了一个年纪比 NCC 还大、内部逻辑比他此刻的脑子还混乱的恐怖怪物。

事情的起因荒诞而日常。几个小时前，原本计划着周末去水库钓鱼、享受几天清净的 NCC，在工位上被项目主管死死按住。主管端着保温杯，语气轻描淡写却透着不容置疑的压迫感：“小南啊，咱们那个老订购网页，又有客户投诉说卡死进不去了。你今晚辛苦一下，把它彻底结了。趁着周末流量小，刚好挂在生产环境观察两天，明天早上你把代码一推，就可以安心过周末了。”
主管拍了拍NCC的肩膀，心安理得地走了。下班时，同事们投来几瞥同情的目光，随后纷纷在打卡机前扫脸，准时消失在电梯口。没过多久，天色彻底暗沉，淅淅沥沥的秋雨很快演变成了砸向玻璃幕墙的瓢泼大雨。

​屏幕上闪烁着刺眼的冷光，Unhandled exception at 0x00411A3C: ... 的字样已经是今晚第十七次弹出了。NCC 猛地灌了一口彻底凉透的咖啡——冰冷得就像窗外倾泻的秋雨，如同冰块一样顺着食道砸进胃里，却浇不灭此刻几乎要烧穿天灵盖的焦躁，也暖不热因绝望而麻木的心。NCC死死盯着代码里一行名为 void* temp_ptr_final_v3 的诡异指针，在心里把起这个天杀名字的实习生祖宗十八代骂了个遍，却只能无奈地顺着调用栈一层层去刨它的祖坟。

NCC 的初衷并不复杂——仅仅是给这个老古董加个并行套接字，接入现代的连接池。但他悲哀地发现，光是理清那些如同癌细胞般增殖的全局变量，把盘根错节的功能解耦，就仿佛是在尝试徒手拆解一枚引信已经严重锈死的未爆弹。无数次想要直接重写的想法被复杂的项目结构和动辄几万代码一个文件的现实给压下去。

这台顶配的 Win11 工作站版主机里，像停尸房一样密密麻麻塞满了各种 Hyper-V 老容器：Linux、Win2000、WinXP、Win7……当然，还有眼前这个为了迎合老旧编译器而苟延残喘的 Win98 虚拟机。每一次哪怕只改动一个字符，NCC 都必须把代码扔回那个 Win98 容器里重新编译。
​漫长而死寂的进度条像钝刀子一样切割着耐心。进度条来到了 90%，风扇的轰鸣声几乎要盖过窗外的雷雨。
​突然，屏幕毫无征兆地卡死。不是 Win98 的蓝屏，而是更底层、更绝望的崩溃——Hyper-V 虚拟机管理控制台直接闪退，宿主机的右下角弹出了一个致命的系统通知：
 **『Hyper-V-VMMS 错误 18560：虚拟机工作进程(vmwp.exe)遇到未处理的异常，虚拟机“Win98_Legacy_Build”已强制断电。错误代码：0xC0000005 (Access Violation)』**

没有蓝屏，没有 dump 文件，像是一个傲慢的少女带着轻视抛出一句“我死机了，但是不告诉你为什么”。刚刚编译了半个小时的进度，瞬间灰飞烟灭。

“草！”NCC也不管当下是深夜，怒吼着捶这个还在装无辜的Windows工作站。“好歹给个具体一点的提示啊，写一个未处理异常是什么东西啊！”，NCC说一句话就捶一下键盘，在工作站的屏幕上打出随机的字符。当 NCC 的拳头再次重重砸向机械键盘的瞬间，指尖刚触碰到回车键，窗外的夜空骤然惨白。一声撕裂空气的炸雷在咫尺处爆响，高压电流瞬间击穿了主板的防浪涌保护。屏幕上的乱码在绝对的高光中扭曲、拉长，NCC 只觉得眼前闪过一片刺眼的 0xFF 纯白，随后，所有的雷声、雨声和风扇的轰鸣，都被彻底的死寂吞没。电灯闪烁几下，再次亮起时，满地只剩下散落的咖啡纸杯，而工位上已经空无一人。
